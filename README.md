# Fire Wizard
A wizard PVP FPS game. Uses P2P.<br>
Uses the Ursina library.<br>
<h3>Controls</h3><br>
Left click: shoot <br>
WASD: walk
<br>
O: exit
<br>
E: unlock mouse<br> I, K: Toggle fullscreen <br> Screenshots in the screenshots folder!
